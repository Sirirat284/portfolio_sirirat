Project Name

Credit Card Approval Prediction

Project Details

Develop a web application and a Machine Learning model to predict the likelihood of a customer being approved or rejected for a credit card application by a bank, using the "credit-card-approval-prediction" dataset. In this project, basic data management techniques such as data cleaning, data transformation, data reduction, and addressing imbalance data issues were employed before training the model.
