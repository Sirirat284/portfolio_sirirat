Credit Card Approval Prediction

Project Name

Credit Card Approval Prediction

Project Details

Develop a web application and a Machine Learning model to predict the likelihood of a customer being approved or rejected for a credit card application by a bank, using the "credit-card-approval-prediction" dataset. In this project, basic data management techniques such as data cleaning, data transformation, data reduction, and addressing imbalance data issues were employed before training the model.

Roles and Project Work Details

Role : Frontend Developer, Machine Learning Model Co-developer, Backend System Consultant

Work Details:

  1. Analyzed data to find insights.
  2. Prepared data through various processes such as cleaning data, handling missing values, transforming data, and addressing data imbalance with smoothing techniques.
  3. Developed web pages for data entry for predictive purposes and other appropriate functions.
  4. Provided advice on backend system development.
