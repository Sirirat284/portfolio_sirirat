Project Name

Develop a model to classify popular dog and cat breeds in Thailand from images

Project Details

Develop a model for image classification using the LinearSVC (Support Vector Machine) model as educational content for a course offered by the company to the general public interested in Data Science. This project involves techniques such as image size reduction to decrease data volume, converting image data into vectors, and training the LinearSVC (SVM) model. Additionally, write a blog on Medium to explain the basic principles for those interested in this type of work, and create a website for users to try out the model via Streamlit.

Roles and Project Work Details

Role : Data Scientist, Project Developer

Work Details:

  1. Researched techniques for image classification.
  2. Prepared data for training and testing the model, including:
    1. Preparing images and labels using the company's library to assist in preparation.
    2. Standardizing image sizes to ensure uniformity.
    3. Converting image data into vectors.
    4. Training the SVM model.
  3. Shared knowledge through writing a blog on Medium to educate peers on this segment.
  4. Utilized the Streamlit library in Python to create a simple web app, suitable for data visualization tasks.
