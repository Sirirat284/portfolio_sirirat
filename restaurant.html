Restaurant and Food Menu Recommendation System

Project Name

Restaurant and Food Menu Recommendation System

Project Details

Create a web application to display all the restaurants within a university campus to solve the problem for outsiders who come for business and do not know what restaurants are available, or for students who are unaware of the dining options on campus. Users should be able to filter restaurants based on their preferences, such as à la carte restaurants, noodle shops, etc., or by religion, like Halal restaurants, which are scarce within the university. The system will show the name, location, and menu of each restaurant, allowing users to make informed decisions, including the ratings of each restaurant.

Roles and Project Work Details

Role : Backend Developer

Work Details:

  2. Designed the database using ER diagrams.
  3. Created a database for data storage using SQL language.
  4. Developed the backend system including selection, search, and filtering functions, such as filtering stores that received 5 stars, based on the requirements from the frontend.
  5. Designed advertising posters.
  6. Provided support for backend tasks.
